# 晨昏之线

时间感知与整点主动问候插件，适用于 [my-neuro](https://github.com/A-night-owl-Rabbit/my-neuro) 框架。

## 功能

### 时间查询工具

- **dawn_dusk_get_time**：用户询问时间、日期、星期时自动调用，支持多时区
- **dawn_dusk_greeting_check**：用户使用"早上好""晚安"等问候语时自动调用，返回当前时间供 AI 判断

### 整点主动问候

在设定时刻（默认 0:00、8:00、12:00、18:00）自动发起问候，并根据对话活跃度智能选择策略：

| 场景 | 判定条件 | 策略 |
|------|---------|------|
| 静默状态 | 距上次交互超过阈值 | AI 直接主动开口问候 |
| 活跃对话中 | 距上次交互很近 | 注入提示词，AI 在下次回复中自然融入 |
| 半活跃状态 | 介于两者之间 | 延迟等待对话间隙再发送 |

## 快速开始

1. 将本插件放入 `plugins/community/dawn-dusk-line/` 目录
2. 在插件管理页面启用插件
3. 根据需要在配置页面调整参数

## 配置说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 问候整点时刻 | 触发主动问候的时刻（24小时制，逗号分隔） | `0,8,12,18` |
| 静默判定阈值 | 距上次交互超过多少分钟视为静默 | 10 分钟 |
| 活跃判定阈值 | 距上次交互小于多少分钟视为活跃 | 3 分钟 |
| 延迟等待超时 | 半活跃状态下最多等多久再放弃 | 30 分钟 |
| 时间检查间隔 | 多少秒检查一次是否到达问候时刻 | 30 秒 |

## 三级策略流程

```
整点到达
  ├─ AI正在说话/处理中？ → 注入提示词（下次回复自然融入）
  ├─ 用户活跃对话中？    → 注入提示词（不打断对话）
  ├─ 用户已静默？        → 直接主动问候
  └─ 半活跃状态？        → 延迟等待
                           ├─ 等到静默 → 发送问候
                           └─ 超时     → 放弃本次
```
